Frequently Asked Questions

Why did you do this?

Well, we all have our passions. Mine takes me to the numbers.

When I watch games, I think of questions. When I think of questions, I try to find the answers. When I find the answers, I share them.

Plus, this was good practice for me. I'm not a web designer, but I wanted to learn some new things.

How did you do this?

Mostly with Python. Some Jinja. Some Github.

For the source data, I'm fetching Paul's Google Spreadsheet

Are you as cool as this looks?

You can't prove I'm not.

How can I help?

I need someone to volunteer to "clean up" some of the source data. Specifically, column E, the cause of death column. If that data were more standardized, I could run an analysis of the death types encountered by Paul.

What's the problem?

For example, some things are listed as 'Stage8' and sometimes 's8'. I could write regexes, and I will if no one helps clean it up, but that will come later.
